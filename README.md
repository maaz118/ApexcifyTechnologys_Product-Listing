# Product Listing Website

## Project Overview
This project is a simple product listing interface built using HTML, CSS and JavaScript.
It allows users to view product details such as title, owner, and price and navigate between different products.
This project was developed during my Web Development Internship at ApexcifyTechnologys.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- 
## Features
- Product title display
- Product owner information
- Product price display
- Next product navigation
- Previous product navigation
- Responsive design layout
  
## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in any web browser
